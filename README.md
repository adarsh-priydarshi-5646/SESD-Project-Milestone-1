# SESD-Project-Milestone-1

## Full Stack Application Project

This repository contains the project documentation and implementation for the Software Engineering and System Design (SESD) course.

### Project Structure

- `idea.md` - Project idea, scope, and key features
- `useCaseDiagram.md` - Use case diagram for the application
- `sequenceDiagram.md` - Sequence diagram showing main flow end-to-end
- `classDiagram.md` - Class diagram with major classes and relationships
- `ErDiagram.md` - ER diagram showing tables and relations

### Focus Areas

- Backend: 75% weightage
- Frontend: 25% weightage

### Engineering Practices

- OOP principles (encapsulation, abstraction, inheritance, polymorphism)
- Clean architecture (controllers/services/repositories)
- Design patterns (applied where appropriate)
- Regular commits and version control
